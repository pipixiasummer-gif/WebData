Cross-Platform Sentiment Analysis Engine

Status: In Development (Beta)

Project Overview

This tool is designed to aggregate public comments from Weibo (微博) and Xiaohongshu (小红书). By analyzing nested comment structures, we generate sentiment heatmaps to help researchers understand the reception of specific topics in finance and education.

Technical Stack

  • Python / Playwright
  • NLP Sentiment Scoring
  • High-Anonymity Proxies
  • JSON Data Containerization

Compliance & Ethics

  • Public Data Access Only
  • Simulated Human Behavior Logic
  • Research-Only Usage
  • No Private Data Scraping
